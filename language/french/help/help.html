Description

Le module wgSitenotice peut être utilisé dans XOOPS

Brève description des fonctionnalités :

  • Ce module créé un site avec mentions légales (marque, vie privée, protection de données, ...)
  • Téléchargez des exemples / thèmes possibles à partir de xoops.wedega.com
  • Téléchargez du site wgSiteNotice à partir d'autres sites internet xoops, si wgSiteNotice y est installé et les droits de téléchargement correctement réglés
  • Les exports des sites sont des thème de base

Installation / Désinstallation

Aucune mesure particulière n'est nécessaire, suivez le processus d'installation standard :
- extraire le dossier wgsitenotice vers le répertoire ../modules
- Installer le module par Admin -> Modules -> Install Module
Des instructions détaillées sur l'installation des modules sont disponibles dans le Manuel d'exploitation de Xoops.

Fonctionnalités

Version de base

L'aide / description est basée sur la version 1.23 beta 1


Préférences du module

Option "Autoriser le téléchargement de vos notes légales" : Vous pouvez définir, si vous voulez permettre à d'autres créateurs de sites xoops de télécharger vos contenus. Attention : si « oui », les versions définies avec les contenus liés seront disponibles en téléchargement (voir aussi l'option "Autoriser le téléchargement").

Option "serveur standard pour la vérification en ligne" : Définit le serveur standard / site internet , celui où vous voulez vérifier les nouvelles versions de notes légales et les télécharger.


Versions

Ici, vous pouvez créer plusieurs versions / sites.
Le classement dans le menu est dans l'ordre des valeurs de "poids".
Chaque version, où l'option "Afficher cette version" est "oui", sera admises dans le menu principal / bloc comme un seul site.
Chaque version, où l'option "Autoriser le téléchargement" est "oui", sera disponible en téléchargement (voir également la préférence du module "Autoriser le téléchargement de vos notes légales".


Contenus

Ici, vous pouvez créer du contenu pour chaque version.
Pour chaque version, vous pouvez créer un ou plusieurs contenus. Le classement des contenus différents sur le site est géré par les valeurs de « poids ».


Versions en ligne

Ici vous pouvez télécharger les versions / sites à partir du site définis dans les préférences du module.
Après avoir cliqué sur "Lancer la vérification des versions disponibles en ligne", vous obtenez une liste de toutes les versions / sites disponibles sur le site internet défini.
Après avoir cliqué sur les versions et les contenus seront téléchargés.


Moyens possibles pour afficher les versions

Vous pouvez lister vos mentions légales sur les pages des utilisateurs de la manière suivante :

  • Menu principal
  • Bloc
  • http://{mondomaine.com}modules/wgsitenotice/index.php?version_id=1 fournit un seul site avec les id de version correspondant (dans ce cas la version id 1)
  • http://{mondomaine.com}modules/wgsitenotice/index.php fournit tous les sites comme un seul site
