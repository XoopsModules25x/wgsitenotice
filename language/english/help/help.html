Help: wgSitenotice

Description

The wgSitenotice module can be used in XOOPS

Short description of the features:

  • This module creates a site with legal notice (imprint, privacy, dataprotection,...)
  • Download of examples/templates possible from xoops.wedega.com
  • Download of wgSiteNotice-sites from other xoops-website, if wgSiteNotice is installed there and the rights to download are set properly
  • Outputs of the sites are template based

Install/uninstall

No special measures necessary, follow the standard installation process:
- extract the wgsitenotice folder into the ../modules directory
- Install the module through Admin -> Modules -> Install Module
Detailed instructions on installing modules are available in the XOOPS Operations Manual.

Features

Basic version

The help/description is based on version 1.23 beta 1


Important module preferences

Option "Allow download of your legal notes": You can define, whether you want to allow other xoops-websitecreators to download your contents. Attention: if "yes" then the defined versions with the related contents will be provided for download (see also option "Allow download").

Option "Standard server for online check": Define the standard server/web site, which you want to check for new versions of legal notes and download them.


Versions

Here you can create several versions/sites.
The ranking in the menu is in order of the values under "weight".
Each version, where the option "Show this version" is "yes", will be listed in the main menu/block as one site.
Each version, where the option "Allow download" is "yes", will be provided for download (see also module preference "Allow download of your legal notes".


Contents

Here you can create the contents for each version.
For each version you can create one or more contents. The ranking of different contents on the site is managed by the the values under "weight".


Online versions

Here you can download the versions/sites from the website define in module preferences.
After clicking on "Start check for available online versions" you get a list of all available versions/sites from the defined website.
After clicking on the versions and contents will be downloaded.


Possible ways to show versions

You can list your legal notices on user pages in following ways:

  • Main menu
  • Block
  • http://{mydomain.com}modules/wgsitenotice/index.php?version_id=1 provides one site with corresponding versions-id (in this case version id 1)
  • http://{mydomain.com}modules/wgsitenotice/index.php provides all sites as one site
