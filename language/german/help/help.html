Hilfe: wgSitenotice

Beschreibung

wgSiteNotice-Modul ist ein Modul für Xoops

Funktionen in Kürze:

  • Das Modul erstellt Seiten mit den wichtigsten rechtlichen Bestimmungen, wie z.B. Impressum, Haftungsausschluss, Datenschutzerklärung
  • Download von Mustern/Vorlagen von xoops.wedega.com möglich
  • Download der wgSiteNotice-Seiten von anderen Xoops-Webseiten, sofern wgSiteNotice installiert ist und die dafür vorgesehenen Rechte entsprechend gesetzt wurden
  • Ausgabe der Seiten templatebasiert

Installation/Deinstallation

Es sind keine speziellen Maßnahmen erforderlich, führen Sie den Standardinstallationsprozess aus:
- Extrahieren Sie den Ordner wgSiteNotice in das Module-Verzeichnis
- Installieren Sie das Modul über Administration -> Module
Wenn Sie zustätzliche Informationen benötigen, lesen Sie die Installationshinweise unter XOOPS Operations Manual.

Funktionen

Zugrundeliegende Version

Der Hilfe/Beschreibung basiert auf der Version 1.23 beta 1


Wichtige Moduleinstellungen

Option "Download Ihrer rechtlichen Hinweise zulassen": Sie können festlegen, ob Sie Ihre Inhalte für andere Xoops-Webseitenersteller freigeben wollen. Achtung: Bei "Ja" werden alle dafür vorgesehenen Versionen und die dazugehörigen Inhalte zum Download freigegeben (siehe auch Option "Download erlauben".

Option "Standard-Server für Überprüfung auf Online-Versionen": Hier können Sie den Standardserver/die Standardwebseite festlegen, die Sie auf aktuelle Versionen überprüfen und von dem/der Sie die Versionen und Inhalte runterladen möchten.


Versionen

Hier können Sie verschiedene Versionen/Seiten anlegen.
Die Reihenfolge im Menü/in der Anzeige erfolgt anhand der Werte unter "Reihung".
Jede Version, deren Option "Version anzeigen" auf "JA" gesetzt ist, wird im Hauptmenü als eigene Seite angeboten.
Jede Version, deren Option "Download erlauben" auf "JA" gesetzt ist, wird für den Download freigegeben (siehe auch Moduleinstellung "Download Ihrer rechtlichen Hinweise zulassen").


Inhalte

Hier können Sie die Inhalte zu den verschiedene Versionen/Seiten erstellen.
Es können pro Version beliebig viele einzelne Inhalte erstellt werden. Die Reihenfolge der Anzeige der einzelnen Inhalte innerhalb einer Version/Seite wird durch die Werte unter Reihung bestimmt.


Online-Versionen

Hier können Sie die Inhalte der Versionen/Seiten von der in den Moduleinstellungen festgelegten Webseite runterladen.
Durch einen Klick auf "Starte Überprüfung auf verfügbare Online-Versionen" werden die auf der entsprechende Webseite verfügbaren Versionen/Seiten aufgelistet.
Durch einen Klick auf können die Versionen und Inhalte runtergeladen werden.


Möglichkeiten zum Anzeigen Ihrer Versionen

Sie können Ihre rechlichen Hinweise auf der Webseite auf folgende Arten anzeigen lassen:

  • Hauptmenü
  • Block
  • http://{mydomain.com}modules/wgsitenotice/index.php?version_id=1 liefert die Seite mit der entsprechende Versions-ID (in diesem Fall von Version ID 1)
  • http://{mydomain.com}modules/wgsitenotice/index.php liefert alle Seiten (Option "Version anzeigen" ist "JA") als eine Seite
